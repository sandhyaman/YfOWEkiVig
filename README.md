# Python Flask 图书管理系统 python205

## 项目概述

本项目是一个基于 Python Flask 和 MySQL 的图书管理系统。系统采用 B/S 架构，前后端分离设计，提供管理员与读者两种角色，实现了图书的增删改查、借阅归还等全流程管理功能。

## 技术栈

- 后端：Python Flask
- 数据库：MySQL
- 前端：HTML/CSS/JavaScript

## 功能模块

### 系统角色

- 管理员
 - 图书增删改查
 - 借阅记录查看
 - 读者信息管理
- 读者
 - 查书
 - 借书
 - 还书
 - 借阅历史查询

### 核心功能

- 库存自动同步
- 超期借阅限制
- 用户认证（含验证码和角色权限控制）

## 运行环境

- Python 环境搭建（推荐使用 PyCharm）
- MySQL 数据库配置

## 部署步骤

1. 准备 Python 环境
2. 安装 `requirements.txt` 中的依赖包
3. 配置 MySQL 数据库
4. 导入数据库脚本
5. 运行 Flask 应用

## 目录结构

```
project/

│
├── databases/ # 数据库脚本目录

│

│
└── requirements.txt # Python 依赖包列表
```
